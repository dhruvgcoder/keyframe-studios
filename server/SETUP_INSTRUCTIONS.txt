📧 EMAIL SETUP NEEDED: Follow these steps to enable email forwarding

┌──────────────────────────────────────────────────────────────┐
│ STEP 1: Open the .env file in the server folder             │
└──────────────────────────────────────────────────────────────┘

Location: server/.env

┌──────────────────────────────────────────────────────────────┐
│ STEP 2: Replace with YOUR email credentials                 │
└──────────────────────────────────────────────────────────────┘

Current .env file has placeholder values:
❌ EMAIL_USER=your-email@gmail.com
❌ EMAIL_PASS=your-app-password

Change to YOUR actual email:
✅ EMAIL_USER=yourname@gmail.com
✅ EMAIL_PASS=abcd efgh ijkl mnop  (16-character app password)

┌──────────────────────────────────────────────────────────────┐
│ STEP 3: Get Gmail App Password (NOT your regular password)  │
└──────────────────────────────────────────────────────────────┘

1. Go to: https://myaccount.google.com/security
2. Enable "2-Step Verification" (if not enabled)
3. Go to: https://myaccount.google.com/apppasswords
4. Select "Mail" app and "Other" device
5. Name it "Contact Form"
6. Google will generate a 16-character password
7. Copy it (remove spaces) and paste in .env as EMAIL_PASS

Example:
If Google shows: abcd efgh ijkl mnop
In .env use: EMAIL_PASS=abcdefghijklmnop

┌──────────────────────────────────────────────────────────────┐
│ STEP 4: Save .env file and restart server                   │
└──────────────────────────────────────────────────────────────┘

In terminal, press Ctrl+C to stop the server, then run:
cd server
node api.js

You should see:
✅ Email server is ready to send messages
